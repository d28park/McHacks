The different states a user skeleton can be found in. The state-machine is: Initializing change to Calibrating, then after user found and calibrated, changes to Tracking. Tracking will change to NotTracking and vice versa when the user exists/re-enter the frame. The only way to return to Calibrating state is to call GeneratorSingleton.Instance.Reset() explicitly.
